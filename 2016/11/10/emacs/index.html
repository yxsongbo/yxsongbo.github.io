markdown 預覽

自本週以來一直在嘗試使用vim來預覽markdown，但vim的插件需要+python，python3，+ruby，Ubuntu下編譯尚不成功，更何况windows版的vim了，今天妥协采用emacs来预览，参看了http://blog.caiyuyu.net/2015/08/29/2015-08-27-[%E6%8A%80%E6%9C%AF%E5%A4%87%E5%BF%98]Windows%E4%B8%8BEmacs+Markdown%E7%9A%84%E4%BD%BF%E7%94%A8/

用法是找到markdown-mode.el(如果存在markdown-mode.elc,先删除，否则会提示el newer than the elc), 在该文件大概900多行的位置，找Command to run makdown,输入上面的命令即可，python 的方式中文乱码，pandoc的方式较好，可是，产生的文件仍然丑陋，所以，makdown预览对我来说还是没有什么价值，还是采用hexo server.
