Hexo 博客

Posted on 2016-10-06

以前的博客是多年前用Jeklly生成的，最近Github提示我在用的markdown解析器将不再支持，按找提示修改了，但用浏览器打开的时候md格式无法解析了，以前我就没有把jeklly搞懂，现在怕麻烦就懒得再去研究，最近看到的博客有一个关键词“hexo“，xiaolai.li写得人人都是工程师教程也以hexo为例子，尝试了下，原来很简单，换个主题就是复制粘贴。

Basic usage：

  1. 新建一篇博客
    hexo new <title>

  2. 生成博客（Generates static files）
    hexo generate
    写了一篇文章后，一开始没有理解这条命令的作用，就使用‘hexo deploy’，git 提示没有文件可以提交，好奇怪啊，不是命名在文件系统中多了一个文件了吗，可能是git的机制是不generate，就认为是不存在的对象。
    这条命令还有一个参数 -d 在成长文件的时候就可以发步到网上，加上这个参数，其实就相当于hexo generate & hexo deploy,不过写的文章应该仔细检查，为了缓冲，还是用：

  3. hexo deploy 发布
